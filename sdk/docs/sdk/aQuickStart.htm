Quick Start Guide

This section describes how to get an image up and running on the Dharma board.

The first step is to install the SDK. The SDK installer is an executable that first installs the Cygwin environment, and then installs the files that make up the Dharma SDK.

For more information about Cygwin, including how to open a bash shell, see http://cygwin.com.

The basic process to creating and loading a Dharma image is:

  1. Build the image
  2. Transfer the image to the device.
