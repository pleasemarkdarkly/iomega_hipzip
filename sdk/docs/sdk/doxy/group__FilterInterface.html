Detailed Description

Dadio(tm) uses an asbtract interface for filters, which allow components to examine and modify the current PCM audio stream. This is the IFilter interface, as defined in ./player/datastream/filter/include/Filter.h. The purpose of this document is to define the IFilter interface and each of its member functions, and to describe their expected behavior in the Dadio(tm) OS.
