Detailed Description

The IMetadata class is used by the Dadio(tm) system to associate a set of key/value pairs with a media item. IMetadata records are created mainly by two sources: the data source during content update, and the media player when a track is set. The content manager provides its own IMetadata subclass, and the media player can be told what type of IMetadata subclass to produce.
