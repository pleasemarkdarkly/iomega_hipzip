Content Manager

Both media files and playlist files are tracked by a content manager in the Dadio(tm) system. The IContentManager interface describes functions for simple addition and retrieval of records. class  IContentRecord
 The base class for both media content records and playlist content records. The content record has an ID unique across the content manager, a URL unique across the system, and the ID of the data source it's stored on. Content records are also considered verified if they've been found to reside on a data source (as opposed to solely existing in the database from a previous run). They also have a status which specifies their ability to be run through a codec for playback. More...

Detailed Description

Both media files and playlist files are tracked by a content manager in the Dadio(tm) system. The IContentManager interface describes functions for simple addition and retrieval of records. The IQueryableContentManager interface adds functions for creating lists of media records based on artist, album, and genre metadata.
